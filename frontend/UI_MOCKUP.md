## Color Reference

### Primary Colors
- **Primary Blue**: `primary-600` (buttons, active states)
- **Light Blue**: `primary-50` (hover backgrounds)

### Status Colors
- **Success**: Green (`green-500` icon, `green-600` text)
- **Error**: Red (`red-500` icon, `red-600` text)
- **Warning**: Yellow (`yellow-500` badge)
- **Info**: Blue (`blue-500` badge)

### Neutral Colors
- **Text**: Gray-900 (dark), Gray-600 (medium), Gray-500 (light)
- **Borders**: Gray-200 (light), Gray-300 (medium)
- **Backgrounds**: White, Gray-50 (subtle), Gray-100 (hover)

### Status Badge Colors
- **Pending**: Yellow background (`yellow-100`), yellow text (`yellow-800`)
- **Processing**: Blue background (`blue-100`), blue text (`blue-800`)
- **Completed**: Green background (`green-100`), green text (`green-800`)
- **Failed**: Red background (`red-100`), red text (`red-800`)
